-SEGUN EL CASO PLANTEADO, SE CREARA COMO CLASE PRINCIPAL DIRECTIVA, EL CUAL ESTE.
-CADA MIEMBRO DE DIRECTIVA(VARIABLE) SERAN LOS DATOS DE LA CLASE PERSONA(ID, NOMRE Y EDAD).
-LA CLASE DIRECTIVA TENDRA UNA LISTA DE LIGAS DE TIPO CLASE LIGA().
-CADA LIGA TENDRA SUS DATOS, EL DATO REPRESENTANTE TENDRA LOS DATOS DE CLASE PERSONA(), UNA LISTA DE EQUIPOS DE TIPO CLASE EQPUIPO() Y UNA LISTA DE TEMPORADAS DE CLASE TEMPORADA().
-CADA TEMPORADA TENDRA UNA LISTA DE PARTIDOS DE CLASE PARTIDO(), CADA PARTIDO TENDRA SU FECHA, EQUPO VISITA, EQUIPO LOCAL Y RESULTADO
-CADA EQUIPO TENDRA SUS DATOS BASES, DATOS DE CAMISETALOCAL Y CAMISETAVISITA TIPO CLASE CAMISETA(); Y UNA LISTA DE JUGADORES TIPO CLASE JUGADOR()
-CADA CAMISETA TENDRA SUS DATOS BASES
-CADA JUGADOR TENDRA SUS DATOS PERSONALES TIPO CLASE PERSONA() Y SUS DATOS COMO JUGADOR.


CLASE DIRECTIVA:-VARIABLE DIRECTORES ES VARIABLE ARRAYLIST TIPO CLASE PERSONA YA QUE SON 5 DIRECTORES POR DIRECTIVA, EL CUAL PODRAN SER MODIFICADOS INDEPENDIENMENTE.
		-VARIABLE LIGAS ES VARIABLE ARRAYLIST TIPO CLASE LIGA YA QUE EXISTEN MAS DE UNA LIGA POR DIRECTIVA(DIRECTIVA ES LA ORGANIZACION DE LA CIUDAD)
		-LAS VARIABLES PRESIDENTE, VICEPRESIDENTE, SECRETARIO, TESORERO SON TIPO CLASE PERSONA GENERANDO ABSTRACCION.

CLASE LIGA:	-VARIABLE REPRESENTANTES ES DE TIPO CLASE PERSONA GENERANDO ABSTRACCION.
		-VARIABLE EQUIPOS ES VARIABLE ARRAYLIST TIPO CLASE EQUIPO YA QUE CADA LIGA ESTA COMPUESTA POR MAS DE UN EQUIPO.
		-VARIABLE TEMPORADAS ES VARIABLE TIPO ARRAYLIST TIPO CLASE TEMPORADA YA QUE CADA LIGA TENDRA UNA O MAS TEMPORADAS.

CLASE TEMPORADA:-VARAIBLE PARTIDOS ES VARAIBLE ARRAYLIST TIPO CLASE PARIDO YA QUE UNA TEMPORADA TIENE VARIOS PARTIDOS.
		-VARIABLE TABLA POSICION ES DE TIPO ARRAYLIST INT PARA ORDENAR LOS EQUIPOS SEGUN PUNTUACION.

-CLASE PARTIDO:	-VISITA ES VARIABLE TIPO CLASE EQUIPO
		-LOCAL ES VARIABLE TIPO CLASE EQUIPO

-CLASE EQUIPO: 	-VARIABLE CAMISETALOCAL ES DE TIPO CLASE CAMISETA PARA MANTENER SUS DATOS ESPECIFICOS SEPARADOS DE VISITA.
		-VARIALBE CAMISETAVISITA ES DE TIPO CLASE CAMISETA PARA MANTENER SUS DATOS ESPECIFICOS SEPARADOS DE LOCAL.
		-VARIABLE JUGADORES ES VARIUABLE ARRAYLIST TIPO CLASE JUGADOR YA QUE UN EQUIPO ESTA COMPUESTA POR MAS DE UN JUGADOR.

-CLASE JUGADOR: -VARIABLE PERSONAL ES DE TIPO PERSONA YA QUE SON DATOS DE UNA PERSONA.
